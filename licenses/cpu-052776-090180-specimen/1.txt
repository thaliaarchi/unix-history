  1.01  Terms in this agreement hereinafter (other than names of parties
and Article headings) which are in capital letters shall have the meanings specified in
the Definitions Appendix.

                            ARTICLE II

            GRANTS OF RIGHTS TO USE OBJECT CODE SOFTWARE

  2.01  WESTERN’S LICENSEE, at the request of the CUSTOMER, grants to
the CUSTOMER, for itself and its SUBSIDIARIES, a personal, nontransferable
and nonexclusive right to use the OBJECT CODE SOFTWARE, solely for its own
or its SUBSIDIARIES’ internal business purposes and solely on DESIGNATED
CPUs.

  No right is granted for the use of the OBJECT CODE SOFTWARE on or in
connection with the operation of any CPU other than a DESIGNATED CPU;
provided however, that the right to use the OBJECT CODE SOFTWARE granted
hereunder shall extend to the temporary use of the OBJECT CODE SOFTWARE
on a single back-up or substitute CPU during any time when a DESIGNATED
CPU is inoperative due to malfunction or the performance of hardware repair,
maintenance or other hardware modifications.
